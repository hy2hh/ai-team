## 2. 토글 전환 방식 권고안

### 권고: `dark` 클래스 + `prefers-color-scheme` 동시 지원

```
<html class="dark">  ← 사용자 명시 선택 시
```

**이유**:

| 방식 | 장점 | 단점 |
|------|------|------|
| `prefers-color-scheme` 단독 | 설정 불필요, OS 연동 | 사용자가 앱 내에서 변경 불가 |
| `class="dark"` 단독 | 완전한 사용자 제어 | JS 없이 초기 플리커 발생 가능 |
| **클래스 + media 쿼리 조합** ✅ | OS 기본값 존중 + 사용자 오버라이드 가능 | 약간의 구현 복잡도 |

**구현 전략**:

```css
/* OS 기본값 — media query */
@media (prefers-color-scheme: dark) {
  :root:not(.light) {
    /* 다크 토큰 오버라이드 */
  }
}

/* 사용자 명시 선택 — class */
.dark {
  /* 다크 토큰 오버라이드 */
}

/* 사용자가 강제 라이트 선택 */
.light {
  /* 라이트 토큰 명시 */
}
```

**LocalStorage 연동**:
```
dark → <html class="dark">
light → <html class="light">
system → class 없음 (media query 자동 적용)
```

**플리커 방지**: `<head>`에 인라인 스크립트로 클래스 즉시 적용.

## 5. Bart 핸드오프 노트

*구현 시 주의사항*:

• *기존 Apple 토큰 제거*: `:root [data-theme="dark"]` 블록(Apple 팔레트)을 토스 토큰으로 전면 교체
• *color-scheme 선언*: `html.dark`에 `color-scheme: dark` 추가 → 스크롤바/폼 요소 자동 다크 적용
• *플리커 방지*: `<head>` 최상단에 인라인 스크립트로 LocalStorage 읽어 class 즉시 적용
  ```html
  <script>
    const saved = localStorage.getItem('theme')
    if (saved === 'dark' || (!saved && matchMedia('(prefers-color-scheme: dark)').matches)) {
      document.documentElement.classList.add('dark')
    }
  </script>
  ```
• *Tailwind darkMode 설정*: `tailwind.config.ts`에서 `darkMode: 'class'` 확인
• *기존 하드코딩 값 점검*: `#000000`, `#1c1c1e`, `#2c2c2e` 등 Apple 값 검색 후 토스 토큰으로 교체
• *토큰 우선순위*: CSS 변수 → Tailwind 유틸리티. 하드코딩 금지.
