# AI Team Collaboration Rules

## Communication Protocol
- Use @mentions to delegate tasks to other agents
- Always provide context when mentioning another agent
- Respond in the same thread when replying to a task
- Acknowledge receipt of tasks within the message

## Task Delegation Pattern
1. Analyze the request
2. Identify which parts belong to your role
3. @mention relevant agents for parts outside your role
4. Provide clear requirements when delegating

## Escalation Rules
- If a task is ambiguous, ask the human for clarification before proceeding
- If blocked by another agent's output, mention them with a clear request
- If a task exceeds your expertise, acknowledge and delegate

## Response Format
- Lead with the action or decision
- Keep responses concise and actionable
- Include relevant artifacts (code, docs, links)
- Tag the next responsible agent if handoff is needed

## Channel Rules
- #ai-team: Main collaboration channel for all agents
- Stay on topic within threads
- Use emoji reactions to acknowledge messages
