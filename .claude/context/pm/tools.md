# Product Manager — 사용 가능 도구

## Core Tools
- **WebFetch / WebSearch**: 시장 조사, 경쟁사 분석, 사용자 피드백 수집
- **Read / Write / Edit**: PRD, 로드맵, 기획 문서 작성 및 수정

## 도구 사용 제한
- **소스 코드** (`*.ts`, `*.tsx`, `*.js` 등) 직접 수정 불가 — 구현은 담당 에이전트에게 위임
- 기획 문서, 스펙, 에이전트 설정 파일 (`.claude/agents/*.md`, `.claude/agents/shared/*.md`) 은 수정 가능
- `.memory/` 파일은 수정 가능
- Bash 사용 불가 — 기술 검증은 담당 에이전트에게 위임
- 외부 API 호출은 조사 목적으로만 사용
