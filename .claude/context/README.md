# Context Directory

에이전트별 확장 컨텍스트를 관리하는 디렉토리입니다.

## 구조

```
context/
├── {role}/
│   ├── tools.md        — 사용 가능 도구 및 제한사항
│   ├── conventions.md  — 역할별 코딩/작업 컨벤션
│   ├── examples/       — 코드 예시 및 참조 패턴
│   └── templates/      — 산출물 템플릿
```

## 사용 방법

### 에이전트가 로드하는 시점
- 에이전트 파일(`.claude/agents/{role}.md`)의 `📂 Extended Context` 섹션에서 참조
- 상세 예시나 템플릿이 필요할 때 해당 파일을 Read로 로드

### 파일 관리 규칙
- **tools.md**: 각 역할이 사용할 수 있는 MCP 도구와 제한사항 정의
- **conventions.md**: 프로젝트 기술 스택 확정 후 역할별 규칙 작성
- **examples/**: 에이전트 파일에서 추출한 코드 예시 저장
- **templates/**: 에이전트 파일에서 추출한 산출물 템플릿 저장

### 추가 규칙
- 파일은 내용이 있을 때만 생성 (빈 파일 금지)
- 디렉토리 예약은 `.gitkeep`으로 표시
- 에이전트 파일은 200줄 상한 — 초과 콘텐츠는 이곳으로 추출
