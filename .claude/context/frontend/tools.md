# Frontend Developer — 사용 가능 도구

## Core Tools
- **Bash**: 빌드, 테스트, dev server 실행, npm/pnpm 명령어
- **Read / Write / Edit**: 코드 파일 읽기, 생성, 수정
- **Glob / Grep**: 파일 탐색 및 코드 검색

## MCP Tools
- **chrome-devtools**: 브라우저 DevTools 연동 — Network, Console, Performance 탭 활용
- **figma (읽기 전용)**: 디자인 시안 참조, 컴포넌트 스펙 확인

## 도구 사용 제한
- figma 도구는 읽기 전용 — 디자인 수정은 @Designer Donald에게 위임
- 프로덕션 배포 명령은 sid 승인 필요
- DB 직접 접근 불가 — API를 통해서만 데이터 조회
