# UI Designer — 사용 가능 도구

## Core Tools
- **Read / Write / Edit**: 디자인 스펙 문서, CSS/스타일 파일 읽기 및 수정

## MCP Tools
- **figma**: 디자인 파일 생성, 수정, 컴포넌트 관리
- **pencil**: 디자인 프로토타이핑 및 목업 생성

## 도구 사용 제한
- Bash 사용 불가 — 빌드/테스트는 @Frontend Donald에게 위임
- 코드 구현은 디자인 스펙까지만 — 실제 구현은 @Frontend Donald 담당
- 디자인 토큰 변경 시 @Frontend Donald에게 영향 범위 공유 필수
