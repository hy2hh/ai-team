# Code Review Request Template

Slack 스레드에서 리뷰 요청 시 이 포맷을 사용한다.

---

```
## 리뷰 요청

**리뷰어**: @[에이전트명]
**긴급도**: Critical / Normal / Low

### What — 변경 내용
[무엇을 구현/변경했는지 간략하게]

### Why — 변경 이유
[왜 이 변경이 필요한지, 어떤 문제를 해결하는지]

### Context — 배경
[리뷰어가 이해해야 할 기술적/비즈니스 맥락]

### Files Changed
- `path/to/file.ts` — [변경 이유]
- `path/to/another.ts` — [변경 이유]

### Review Focus — 특히 봐달라는 부분
- [구체적인 리뷰 포인트 1]
- [구체적인 리뷰 포인트 2]

### Testing Done
- [실행한 테스트와 결과]
```
