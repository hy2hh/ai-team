팀 컨벤션에 맞춰 작성했습니다:

```tsx
import React from 'react';

interface ActionButtonProps {
  /** 버튼에 표시할 텍스트 */
  label: string;
  /** 버튼 클릭 시 실행될 핸들러 */
  onClick: () => void;
  /** 버튼 비활성화 여부 */
  disabled?: boolean;
}

/**
 * 클릭 핸들러가 있는 기본 액션 버튼 컴포넌트
 *
 * @param label - 버튼 텍스트
 * @param onClick - 클릭 이벤트 핸들러
 * @param disabled - 비활성화 여부 (기본값: false)
 */
function ActionButton({ label, onClick, disabled = false }: ActionButtonProps) {
  const handleClick = () => {
    onClick();
  };

  return (
    <button
      onClick={handleClick}
      disabled={disabled}
      type="button"
    >
      {label}
    </button>
  );
}

export default React.memo(ActionButton);
```

**사용 예시:**

```tsx
<ActionButton
  label="제출"
  onClick={() => console.log('클릭됨!')}
/>
```

팀 컨벤션 체크리스트:
- ✅ `if`/`else` 브레이스 (해당 없음)
- ✅ 세미콜론, 싱글 쿼트, 2-스페이스 들여쓰기
- ✅ 이벤트 핸들러 `handleClick` 네이밍
- ✅ 이벤트 prop `onClick` 네이밍
- ✅ `React.memo` 적용
- ✅ TSDoc 작성
- ✅ `any` 없음
