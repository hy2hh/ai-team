팀 컨벤션(team-convention.md) 기준으로 작성합니다:

```tsx
import React, { useCallback } from 'react';

interface Props {
  /** 버튼 레이블 */
  label: string;
  /** 버튼 클릭 시 호출되는 핸들러 */
  onClick: () => void;
}

/**
 * 클릭 이벤트 핸들러가 달린 기본 버튼 컴포넌트
 */
export const ActionButton = React.memo(({ label, onClick }: Props) => {
  const handleClick = useCallback(() => {
    onClick();
  }, [onClick]);

  return (
    <button type="button" onClick={handleClick}>
      {label}
    </button>
  );
});

ActionButton.displayName = 'ActionButton';
```

**사용 예시:**

```tsx
const Parent = () => {
  const handleButtonClick = () => {
    console.log('버튼 클릭됨!');
  };

  return (
    <ActionButton
      label="클릭하세요"
      onClick={handleButtonClick}
    />
  );
};
```

**적용한 컨벤션 포인트:**
- `React.memo` — 불필요한 리렌더 방지
- `useCallback` — onClick 레퍼런스 안정화
- `handleClick` / `onEvent` 핸들러 명명 규칙 준수
- `TSDoc` — Props 인터페이스 및 컴포넌트에 문서화
- `displayName` — React DevTools에서 컴포넌트 식별 용이
- 세미콜론, 싱글 쿼트, trailing comma 등 Prettier 규칙 준수

스타일링이나 추가 기능(disabled 상태, 로딩 스피너 등)이 필요하면 말씀해 주세요!
