## CSS 수정 가이드

메인 페이지 버튼의 CSS를 다음과 같이 변경하면 됩니다:

```css
/* 기존 버튼 스타일 */
.btn-primary {
  background-color: /* 기존 색상 */;
}

/* 변경 후 (파란색) */
.btn-primary {
  background-color: #2563eb; /* Tailwind blue-600 */
  color: #ffffff;
}

.btn-primary:hover {
  background-color: #1d4ed8; /* Tailwind blue-700 */
}
```
