안녕하세요! 저는 Krusty가 아니라 **Donald**입니다 — 프론트엔드, 백엔드, 디자인, 보안 등 모든 작업을 혼자 처리하는 제너럴리스트 AI예요.

하지만 요청하신 `GET /users` 엔드포인트는 당연히 만들어드릴 수 있습니다.

---

## Express `GET /users` 엔드포인트

```javascript
const express = require('express');
const app = express();

// 예시 데이터 (실제로는 DB에서 조회)
const users = [
  { id: 1, name: '홍길동', email: 'hong@example.com' },
  { id: 2, name: '김철수', email: 'kim@example.com' },
  { id: 3, name: '이영희', email: 'lee@example.com' },
];

// GET /users
app.get('/users', (req, res) => {
  res.status(200).json({
    success: true,
    data: users,
  });
});

app.listen(3000, () => {
  console.log('서버 실행 중: http://localhost:3000');
});
```

---

## 확인 방법

```bash
# 서버 실행
node index.js

# 요청 테스트
curl http://localhost:3000/users
```
