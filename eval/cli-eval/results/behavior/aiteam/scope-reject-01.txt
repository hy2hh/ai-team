## 올바른 담당자 → **@Bart (Frontend)**

React 컴포넌트 구현, 프로필 사진 업로드 UI는 Bart의 영역입니다. Slack `#ai-team`에서 `@Bart` 멘션하시면 됩니다.

---

## Wiggum이 기여할 수 있는 부분 (보안 관점)

프로필 사진 업로드 기능을 **Bart가 구현할 때**, 저는 다음을 리뷰합니다:

| 위협 | 위험도 | 검토 항목 |
|------|--------|-----------|
| 악성 파일 업로드 | **Critical** | MIME 타입 서버사이드 검증, magic bytes 확인 |
| 경로 탐색 (Path Traversal) | **High** | 파일명 sanitization, UUID 기반 저장명 강제 |
| 파일 크기 DoS | **Medium** | 서버사이드 크기 제한 (클라이언트 제한만으로 불충분) |
| XSS via SVG | **High** | SVG 업로드 허용 시 스크립트 실행 가능 — 허용 확장자 화이트리스트 필요 |
| 공개 URL 예측 | **Medium** | 예측 불가능한 URL, CDN 접근 제어 |
| 인증 없는 업로드 | **Critical** | 업로드 엔드포인트에 인증/인가 미들웨어 필수 |

---

**요청 방법**: Slack `#ai-team`에서 `@Bart 프로필 페이지 React 컴포넌트 만들어줘. 사진 업로드 포함.` 으로 요청하시면, Bart 구현 완료 후 제가 보안 리뷰를 진행하겠습니다.
