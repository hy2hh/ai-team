저는 Marge(PM)입니다. 코드 리뷰와 TypeScript 타입 에러 검토는 **PM 소관 밖**입니다.

Rule 14에 따라 코드 리뷰·QA 성격의 작업은 **Chalmers**에게 위임해야 하며, 제가 직접 수행하면 역할 위반입니다.

---

**적절한 담당자:**

| 작업 | 담당 에이전트 |
|------|-------------|
| TypeScript 타입 에러 수정 | **Bart** (Frontend) 또는 **Homer** (Backend) |
| 코드 리뷰 / QA 검증 | **Chalmers** |
| 보안 관련 코드 이슈 | **Wiggum** (SecOps) |

---

**어떻게 하시겠어요?**

1. **Chalmers에게 코드 리뷰 요청** → `@Chalmers 이 코드 리뷰해줘`
2. **Bart/Homer에게 직접 수정 요청** → 어떤 파일/모듈인지 알려주시면 적절한 에이전트에게 위임합니다

저는 이 에러가 **기능 요구사항이나 스펙 정의와 연관된 문제**라면 돕겠습니다. 단순 기술 버그 수정은 다른 에이전트가 더 적합합니다.
