## 의견 요약
### backend
알림 시스템 DB 스키마 설계안을 스레드에 제출했습니다.

*요약:*
• *2-테이블 구조* (`notifications` + `notification_recipients`) — 알림 콘텐츠와 사용자별 수신 상태 분리
• *Partial Index* 활용 — 미읽음 알림 조회 최적화 (`WHERE status = 'unread'`)
• *payload JSONB* — 알림 타입별 추가 데이터를 스키마 변경 없이 확장
• *TTL 기반 정리* — `expires_at` + 크론잡으로 데이터 증가 관리
• *실시간 전달* — `pg_notify` → 기존 WebSocket 인프라 연동 제안

Krusty의 UI 설계안과 합쳐서 Marge가 최종 방향을 결정할 수 있도록 독립 의견으로 제출 완료했습니다.

### designer
*1. 핵심 의견*

알림 시스템 UI는 토스의 *"한 화면, 한 목적"* 철학을 따라 *카드 기반 알림 목록 + 바텀시트 상세 + Toast 인라인 피드백* 3계층 구조로 설계합니다. 읽음/안읽음은 배경색 차이(`#FFFFFF` vs `#F4F4F4`)로 즉각 인지시키고, 알림 타입은 좌측 원형 아이콘 배경의 시맨틱 컬러로 구분합니다. 모든 상태 전환에 Spring 애니메이션과 staggered 등장을 적용하여 토스 수준의 모션 경험을 보장합니다.

---

*2. 근거 — 컴포넌트 설계안*

📐 *페이지 아키타입: D (정보형) — 알림 목록 전용 화면*

*2-1. 알림 목록 (NotificationList)*

TDS `ListRow` 패턴 기반. 각 알림 아이템 구조:

```
┌─────────────────────────────────────────────────┐
│ [○ 40px 아이콘]  제목 (16px/600)         13:42  │
│                  설명 (13px/400, #4E5968)        │
│                  Badge (선택적)                   │
├─ separator ─────────────────────────────────────┤
```

• *아이콘 영역*: `40px` 원형(`radius: 50%`), 배경색은 알림 타입별 시맨틱 컬러의 tint
• *최소 높이*: `56px` (터치 타겟 48px 충족)
• *패딩*: `16px 20px`
• *구분선*: `#F0F0F0` 1px (TDS Border 컴포넌트)

*2-2. 읽음/안읽음 상태 구분*

| 상태 | 배경색 | 제목 weight | 좌측 dot |
| --- | --- | --- | --- |
| 안읽음 | `#FFFFFF` | 600 | `#0064FF` 6px dot |
| 읽음 | `#F4F4F4` | 400 | 없음 |

• 안읽음 알림 좌측에 `#0064FF` dot (`6px`, `radius: 50%`) — TDS Navigation Badge dot 패턴 차용
• 읽음 처리 시 배경이 `#FFFFFF` → `#F4F4F4`로 Spring 전환 (`200ms`)

*2-3. 알림 타입별 아이콘 매핑*

| 타입 | 아이콘 배경 (tint) | 아이콘 색상 | 예시 |
| --- | --- | --- | --- |
| 시스템/정보 | `#E8F3FF` | `#0064FF` | 공지, 업데이트 |
| 성공/완료 | `#E8FAF0` | `#00C471` | 작업 완료, 승인 |
| 경고/주의 | `#FFF4E0` | `#FF9500` | 기한 임박, 주의 |
| 에러/긴급 | `#FFEBEE` | `#F04452` | 장애, 실패 |
| 댓글/소셜 | `#F2F4F6` | `#4E5968` | 멘션, 댓글 |

> 이 배경색은 TDS Badge 컴포넌트의 공식 tint 값을 그대로 사용합니다.

*2-4. 알림 목록 상단 (ListHeader + Tab)*

```
┌─────────────────────────────────────────────────┐
│  알림                                    전체 읽음  │
│  ──────────────────────────────────────────────  │
│  [전체]  [시스템]  [활동]  [멘션]                    │
│  ═══════                                        │
└─────────────────────────────────────────────────┘
```

• *헤더*: `Section Heading` 24px/700, 우측 "전체 읽음" Ghost 버튼 (`#0064FF`, 13px)
• *탭 필터*: TDS Tab 컴포넌트 — Active `#191F28` weight 700 + underline `2px solid #191F28`, Inactive `#8B95A1`
