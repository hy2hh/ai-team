# AI 코딩 에이전트 시장 조사 (2026-04-10)

## 시장 규모
- 2025년: $7~8B (기관별 상이: Gartner $3.5B, SNS Insider $4.7B, MarketsandMarkets $8.1B)
- 2026년 예측: $8.5~10B
- 2032년 예측: $30~127B (CAGR 27~48%)

## 주요 플레이어

| 도구 | ARR/MS | 특징 | 가격 |
|------|--------|------|------|
| Cursor (Anysphere) | $2B ARR, 18% MS | Agent-first IDE, 멀티파일 | $20/mo Pro |

| Lovable | $200M ARR (→$1B 예측) | Vibe Coding, 비개발자 타겟 | - |

## 2026 핵심 트렌드
1. 멀티에이전트 시스템 프로덕션 진입 (오케스트레이터 + 전문 에이전트)
2. 자율 실행 루프 (분~시간 단위 장시간 실행)
3. Vibe Coding: 자연어로 앱 개발
4. 개발자 역할 전환: 코드 작성 → 에이전트 오케스트레이션
5. 비용 효율성 경쟁 심화

## 채택 지표
- AI 도구 사용 개발자: 84~85%
- 엔터프라이즈 팀 AI 사용: 90% (2025 말)
- AI 생성 코드 비율: 41% (2025), 26.9% 프로덕션 코드
- 주당 평균 절약 시간: 3.6시간/개발자

## 리스크
- AI 생성 코드 보안 취약점: 48%
- AI 고채택팀 PR 리뷰 시간 91% 증가

## 우리 포지셔닝 (ai-team 맥락)
- 우리는 멀티에이전트 오케스트레이션 트렌드와 완전 정렬
- Slack 기반 협업은 차별화 포인트 (대부분 IDE 중심)
- 칸반 보드 + AI 에이전트 통합은 신규 카테고리 가능성
